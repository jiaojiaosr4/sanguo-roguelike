# 🏯 三国肉鸽 - Three Kingdoms Roguelike

一款以三国时期为背景的回合制 Roguelike 地牢探险游戏，在浏览器中即可游玩。

## 🎮 游戏特色

- **四位传奇英雄**：选择刘备、关羽、曹操或诸葛亮，每位英雄拥有独特的技能
- **程序化生成地牢**：每次游戏地图随机生成，房间、走廊、敌人和物品分布各不相同
- **回合制战斗**：经典 Roguelike 玩法，敌我轮流行动
- **战争迷雾**：探索未知区域，发现隐藏的宝物和敌人
- **10层关卡**：层层递进，每层敌人更强，每2层有Boss镇守
- **五部位装备系统**：头饰、衣服、裤子、兵器、坐骑，Boss掉落专属神装
- **历史名物**：紫金雉尾冠、黄金锁子甲、虎皮战裙、赤兔马、方天画戟
- **消耗品系统**：草药、金疮药、战神丹等助力战斗
- **升级成长**：击杀敌人获取经验，提升等级增强属性

## 🚀 快速开始

### 方式一：直接打开（推荐）

1. 下载项目文件
2. 用浏览器打开 `index.html`
3. 选择英雄，开始游戏！

### 方式二：GitHub Pages

将项目推送到 GitHub 仓库，在 Settings → Pages 中启用 GitHub Pages。

### 方式三：本地服务器

```bash
# 使用 Python
python -m http.server 8080

# 使用 Node.js
npx serve .
```

然后在浏览器中访问 `http://localhost:8080`

## 🏗️ 技术栈

- 纯 HTML/CSS/JavaScript，无需任何依赖
- Canvas 渲染游戏画面
- 程序化地图生成（BSP + 走廊连接）
- 射线投射视野算法（FOV/Fog of War）
- 响应式设计，支持桌面和移动端
